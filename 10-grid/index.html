grid system

  • display속성이 grid인 컨테이너는 block레벨과 유사함.
    grid 컨테이너의 자식요소는 기존의 display속성이 사라짐.
  • grid 컨테이너의 격자 설정
    grid-template-rows: repeat(갯수, 한줄의높이(auto,1fr))
    grid-template-columns: repeat(갯수, 한칸의넓이(auto,1fr))
    gap : 그리드 간의 여백 설정
  • grid를 설정
    grid-row: 2 / 6 ;(2번째줄에서 시작해서 5번째줄에서 끝난다)
    grid-column: 2 / span1 ; (2번째칸에서 시작하는 1칸짜리)
    grid-area: 2 / 3 / 7 / 4 ; (2번째 줄에서 시작 6번째 줄에서 끝나면서, 3번째 칸에서 시작해서 3번째 칸에서 끝난다)
