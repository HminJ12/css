flex

자식요소의 레이아웃을 설계하는 목적임 flex요소의 자식요소는 display속성이 유효하지 않는다

flex의 종류

  • display:flex;

    • 부모요소는 block레벨과 동일한 성격을 갖는다
    • 자식들의 크기가 부모의 크기와 동일해진다

  • display:inline-flex;

    • 부모의 크기가 자식의 크기에 의해 설정된다
    • 부모의 크기를 지정하지 않는다
    • 자식의 크기를 %로 지정하지 않는다
    • 수평레이아웃에 사용한다

  • column, column-reverse (자식요소들은 넓이가 동일해짐)

flex속성

부모요소가 flex속성일 경우 자식요소에게 넓이를 설정하는 속성(자식한테 준다)

  • 크기를 자동으로 설정
  • 축이 row일 경우 넓이설정, 축이 col일 경우 높이설정
  • flex속성의 값이 설정되면 크기를 지정할 수 없음
  • flex계산요령
    자식요소가 3개일 경우, 1,1,2를 지정할 경우 (2/(1+1+2))*100 자식요소들끼리 계산할 때 flex속성을 사용함
